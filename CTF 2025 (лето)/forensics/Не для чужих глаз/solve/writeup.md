# Не для чужих глаз - writeup

## Идея
В дампе много лишнего трафика. Полезные данные спрятаны в DNS запросах, которые уходят на публичный DNS сервер 8.8.8.8. В имени каждого запроса первый поддомен - это небольшой кусок hex. Если собрать эти куски по порядку, получится hex строка, которая после декодирования превращается в base64, а затем - во флаг.

## Решение через Wireshark
1. Открыть `not_for_your_eyes.pcapng`.
2. Поставить фильтр:
   - `dns && ip.dst == 8.8.8.8`
3. В списке пакетов будет серия DNS запросов вида `<hex>.example.com`.
4. Взять первый поддомен каждого запроса по порядку и склеить в одну строку.

Получится:

```
526b564456455a37626a423058334d775879517a513356534d33303d
```

5. Декодировать hex в байты, получится ASCII строка:

```
RkVDVEZ7bjB0X3MwXyQzQ3VSM30=
```

6. Это base64. Декодируем base64 и получаем флаг.

## Вариант без Wireshark (Python)

```python
import base64

hex_s = "526b564456455a37626a423058334d775879517a513356534d33303d"
raw = bytes.fromhex(hex_s)
print(raw.decode())
print(base64.b64decode(raw).decode())
```

Результат:

```
FECTF{n0t_s0_$3CuR3}
```
