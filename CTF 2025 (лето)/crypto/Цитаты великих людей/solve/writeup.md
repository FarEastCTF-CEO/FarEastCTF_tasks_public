# Решение

    В файле `Blaise-Crypto.txt` дано 6 строк по 32 символа. Алфавит - `0-9a-z`.
    По смыслу задания и названию файла можно догадаться, что используется шифр Виженера с ключом `crypto`.

    ## Шаг 1. Расшифровать Виженер

    Удобно работать с числами:
    - символы строки переводим в base36 (`0-9a-z` -> 0..35)
    - сдвиг для символа ключа берем как `a=0 ... z=25`
    - после вычитания берём остаток по 16, чтобы вернуться к hex-символам `0-9a-f`

    Пример скрипта:

    ```python
    base36 = "0123456789abcdefghijklmnopqrstuvwxyz"
    hexabc = "0123456789abcdef"
    key = "crypto"

    def dec_line(line: str) -> str:
        out = []
        for i, ch in enumerate(line.strip()):
            c = base36.index(ch)
            shift = ord(key[i % len(key)]) - ord("a")  # 0..25
            out.append(hexabc[(c - shift) % 16])
        return "".join(out)

    with open("Blaise-Crypto.txt", "r", encoding="utf-8") as f:
        for line in f:
            print(dec_line(line))
    ```

    После расшифровки получаем MD5 хэши:

    - `640a936dc45b4c732df0c9850140140f`
- `27d55eca2364185d830ddea2f03508ae`
- `6899572b4be3e7990caeb91796e83fa3`
- `1ee9579f46b8426995d13260a016a203`
- `1214306fa186e44424b516666f5023eb`
- `6859a516c11314a8a59563be1867cd0a`

    ## Шаг 2. Развернуть MD5 в текст

    Эти 6 хэшей - это части цитаты. Любым MD5 reverse сервисом (например, через публичные базы) разворачиваем каждый хэш в исходную строку, собираем цитату и определяем автора.
