# Информация
В pcap есть HTTP ответ с бинарным телом (32 байта) и заголовком X-Hint. Ключ - MD5 от строки из подсказки, расшифровка AES-ECB.

# Выдать участникам

# Описание задачи
Мы перехватили трафик одного сервера. Вроде бы обычный HTTP, но в ответе на один из запросов вместо текста — какая-то бинарная белиберда. Возможно, сервер что-то скрывает? Мы смогли узнать только то, что ответ зашифрован каким-то стандартным алгоритмом.
