# Информация
Есть API, которое принимает HTML и генерирует PDF через gotenberg. Все складывается во временные директории, которые быстро очищаются. Внутренний бот каждые 5 секунд генерирует PDF из HTML с флагом. Нужно через iframe с file:// заглянуть во внутренний /tmp gotenberg и успеть прочитать HTML с флагом.

# Описание задачи
Флаг существует. Он появляется. Он исчезает. Как мираж. Лови момент, пока он не испарился. Один сервис стабильно каждые 5 секунд создает секретный PDF из HTML-шаблона с флагом.
