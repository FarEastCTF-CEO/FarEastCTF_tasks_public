# Категория | Имя задачи
Web | IDOR

# Информация
Flask-приложение с уязвимостью IDOR: доступ к данным пользователей по перебору идентификатора.

# Деплой
cd deploy
docker compose up --build

# Описание задачи
В веб-приложении профили пользователей открываются по идентификатору в URL. Найдите способ получить доступ к скрытому аккаунту и забрать флаг.

# Флаг
FECTF{idor_can_be_tricky}
