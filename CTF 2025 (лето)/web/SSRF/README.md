# Категория | Имя задачи
Web | SSRF

# Информация
Сервер делает запрос по URL, заданному пользователем. Возможна SSRF на localhost.

# Деплой
cd deploy
docker compose up --build

Открыть: http://localhost:8080

# Выдать участникам
- ip инстанции

# Описание задачи
Приложение умеет загружать ресурсы по ссылке. Используйте SSRF, чтобы добраться до внутренних эндпоинтов и получить флаг.

# Флаг
FECTF{ssrf_is_not_always_rce}
