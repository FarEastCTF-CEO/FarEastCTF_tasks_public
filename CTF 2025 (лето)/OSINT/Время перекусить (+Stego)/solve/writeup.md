# Решение

1. Смотрим метаданные `CTF_pic1.jpg`. В EXIF (XPSubject) лежит строка `pravilniy_postroit`.
2. Извлекаем сообщение из `CTF_pic2.png` методом LSB. Получаем `treugolnik`.

Подсказка складывается в фразу: `pravilniy treugolnik postroit`.

3. По фото находим места съемки (поиск по картинке). Оба здания находятся в Хабаровске, точки считаем вершинами.
4. Строим правильный треугольник по двум найденным вершинам и определяем третью точку.
5. В третьей точке находится ресторан **Vdrova**.

## Флаг

`FECTF{Vdrova}`
