# Категория | Имя задачи
OSINT | В поисках ушастика

# Информация
OSINT: декодировать Base64, найти город и аттракцион по сайту и карте.

# Описание задачи
Городские легенды гласят, что в одном из парков России прячется таинственный ушастый аттракцион, названный в честь существа из забытых сказок. Говорят, его имя шепчут в переулках старого театра, а координаты зашифрованы в цифровом послании, брошенном словно вызов детективам.
Перед вами загадочная строка:
aHR0cHM6Ly93d3cuY2hlaG92c2t5LnJ1LyM=
Она - ключ к началу квеста. Расшифруйте её, следуйте по следам, и вы найдете город, где когда-то гулял великий писатель, и аттракцион, чьи уши слышат шелест флага...
