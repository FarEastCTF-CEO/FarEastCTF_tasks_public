# Решение

1. Декодируем строку Base64:

`aHR0cHM6Ly93d3cuY2hlaG92c2t5LnJ1LyM=`

Получаем ссылку на сайт.
2. По сайту определяем город и находим театр.
3. По карте рядом с театром находим парк и аттракционы.
4. Среди аттракционов подходит только один вариант с "большими ушами" - **Чебурашка**.
5. Собираем флаг в формате из задания: `<аттракцион>-<город>`.

## Флаг

`FECTF{cheburashka-taganrog}`
