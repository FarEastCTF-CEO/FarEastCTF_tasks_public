# Информация
OSINT: вычисления по координатам 7 чудес света, затем поиск места по итоговым координатам.

# Описание задачи
Мы решили с друзьями устроить кругосветное путешествие, но один из нас предложил одно место на Родине.
Вместо конкретного названия он написал следующее:
"Найти, сложить, и вычесть 182°51′54′′ с.ш. 179°46′44′′ в.д.,
и в этом вся наука
чтоб цифры отыскать друзья, нужны вам будут чудеса,
не бойтесь, здесь не туго’
Никудышный из него поэт...
Найденный флаг записать в формате FECTF{Flag} В ОДНО АНГЛИЙСКОЕ СЛОВО, МАЛЕНЬКИМИ БУКВАМИ.
