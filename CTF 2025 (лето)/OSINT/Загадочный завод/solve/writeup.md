# Решение

1. По фото из вложения определяем завод (поиск по картинке).
2. По заводу выясняем, какие модели машин с него сходили. В решении подходит **Москвич 408**.
3. Ищем информацию о первом автомобиле известного артиста и находим, что первым артистом, владевшим Москвич 408, был Михаил Шуфутинский.
4. Самая узнаваемая песня - "The Third of September".
5. По формату задания: нижний регистр, без пробелов.
