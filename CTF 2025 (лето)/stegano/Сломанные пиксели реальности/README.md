# Информация
В IHDR PNG занижена высота. Если увеличить высоту до фактического количества строк, внизу появится строка base58, которая декодируется в флаг.

# Выдать участникам

# Описание задачи
Вам передали артефакт - изображение, которое, кажется, искажает саму реальность. Его создатель утверждал, что «границы картинки - это иллюзия», но никто не понял, что он имел в виду... до сегодня. Присмотритесь: байты высоты файла содержат координаты цифровой пропасти, где пиксели сливаются в тайное послание. Измените их, и внизу, словно надпись на стене заброшенного сервера, проступит строка, закодированная в языке, который понимают только те, кто знает, что «58» - это не просто число...
