# Информация
Два слоя сокрытия: в JPG спрятан RAR-архив с подсказкой (шифр Цезаря), а в PNG через OpenPuff спрятан файл с флагом.

# Выдать участникам
- house.jpg
- field.png

# Описание задачи
Вам переданы две загадочные картинки - house.jpg и field.png. Кажется, они просто показывают дом и поле, но это иллюзия. Их создатель использовал два слоя сокрытия:

house.jpg - это не фото, а RARJPEG-контейнер, внутри которого архив с шифрованным посланием.

field.png - здесь флаг спрятан в самом ландшафте, как зерно в почве, через LSB-стеганографию.
