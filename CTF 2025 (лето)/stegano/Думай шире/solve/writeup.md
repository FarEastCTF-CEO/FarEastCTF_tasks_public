# Решение

## 1. Находим пароль к архиву

Файл `Think_Wider.jpg` - это RARJPEG: в конце JPG спрятан RAR-архив.

Пароль собирается из двух частей:
1) площадь изображения (Width * Height)
2) строка из метаданных (XMP), где лежит `+HwJ4z` (плюс не нужен)

Для картинки 1024x724:
```text
1024 * 724 = 741376
```

Итого пароль:
```text
741376HwJ4z
```

## 2. Извлекаем архив

Откройте JPG в WinRAR и введите пароль, внутри будет файл `untitled` (SVG).

## 3. Достаём флаг из SVG

Откройте SVG любым просмотрщиком или браузером. Флаг находится на пересечении фигур (подсказка: используйте выделение и поиск текста).

**Флаг:** `FECTF{Bye_People}`
