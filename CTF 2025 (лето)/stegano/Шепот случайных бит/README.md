# Информация
PNG содержит скрытый файл, извлекаемый утилитой OpenPuff с тремя паролями из имени файла. Внутри - бинарная строка, которую нужно перевести в байты и получить флаг.

# Выдать участникам
- OpenPuf_12345678_12345678_12345678.png

# Описание задачи
Вам передан файл-загадка: «OpenPuf_12345678_12345678_12345678». Его размер намекает, что внутри спрятано больше, чем видит глаз. Но обычные инструменты слепы к этой тайне. Здесь правит бал стеганография, а пиксели - не просто цвета, а носители битовой магии. Обратите внимание: самые яркие и тёмные участки изображения ведут себя... подозрительно. Как будто кто-то нарисовал криптографический граффити поверх сжатия.
