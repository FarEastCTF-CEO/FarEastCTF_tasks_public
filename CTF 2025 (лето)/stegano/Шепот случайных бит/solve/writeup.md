1. По названию файла видно три одинаковых пароля: `12345678_12345678_12345678`.
2. Откройте утилиту **OpenPuff** -> **Unhide**.
3. В качестве контейнера выберите `OpenPuf_12345678_12345678_12345678.png` и введите три пароля: `12345678`, `12345678`, `12345678`.
4. После извлечения получится файл `somecode.txt` с бинарной строкой (последовательность 0 и 1).
5. Переведите бинарную строку в байты (по 8 бит) и декодируйте как UTF-8, например так:

```python
bits = open("somecode.txt", "r", encoding="utf-8").read().strip()
data = bytes(int(bits[i:i+8], 2) for i in range(0, len(bits), 8))
print(data.decode("utf-8"))
```
