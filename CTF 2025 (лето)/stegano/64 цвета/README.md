# Категория | Имя задачи
Stegano | 64 цвета

# Информация
Картинка 8x8. В каждой строке два оттенка: светлый и чуть темнее. Темный = 1, светлый = 0. 8 строк по 8 бит дают 8 ASCII символов.

# Выдать участникам
- Colors.png

# Описание задачи
Перед вами - цифровая мозаика из 64 пикселей, где каждый цвет шепчет биты. Изображение кажется простым, но его создатель спрятал здесь послание, зашифрованное в языке оттенков. Светлые точки - это тишина (0), а чуть более тёмные - импульсы (1). Ваша задача - расшифровать танец цветов, построив бинарные мосты между пикселями, и услышать, что скрыто в их мерцании.
