# Решение

1. Открываем `Sudoku.xlsx`.
2. В правой части таблица соответствия числа -> символ:
   - 1 -> T
   - 2 -> F
   - 3 -> C
   - 4 -> L
   - 5 -> g
   - 6 -> f
   - 7 -> @
   - 8 -> {
   - 9 -> }
3. Решаем судоку с учетом этой подстановки. Готовое решение лежит в `Sudoku_Solution.xlsx` (для организаторов).
4. В выделенных (желтых) ячейках получаем числа и заменяем их на символы по таблице.
5. Собираем строку `gL@FCTf` и оформляем флаг.

## Флаг

`FECTF{gL@FCTf}`
