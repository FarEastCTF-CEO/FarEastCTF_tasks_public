# Решение

## Идея
В бинаре есть проверка пароля. Строка пароля лежит в данных и легко находится статически.

## Шаги
1. Достаем строки:
```bash
strings -n 3 task.exe | grep -i v3ry
```

2. Находим пароль:
```
v3ry_v3ry_s3cr3t
```

3. Запускаем бинарь в Windows или через Wine и вводим пароль. После успешной проверки приложение выводит флаг.

## Флаг
`FECTF{Is_rEv_Fl@g!}`
