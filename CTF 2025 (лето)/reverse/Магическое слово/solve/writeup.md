# Решение

## Идея
Программа проверяет введенную строку. Внутри она собирает три 32-битных значения и сравнивает их с результатами простых арифметических операций с константой `0xcafedead`.
Если отменить эти операции, можно восстановить исходные байты слова.

## Восстановление
В коде встречается:
- `local6 = 0xcafedead`
- и три значения, которые получаются как `X - local6`

Значит исходные значения:
- `first = 0x1114f2dff - local6`
- `second = 0x10b421402 - local6`
- `third = 0xcb431201 - local6`

Далее эти 32-битные числа разбиваются на байты (little endian) и склеиваются в строку.

Результат:
`ROPFU5C@T3D`

## Проверка
```bash
printf "%s\n" "ROPFU5C@T3D" | ./task200
```

Выводит `RIGHT!`.

## Флаг
`FECTF{ROPFU5C@T3D}`
