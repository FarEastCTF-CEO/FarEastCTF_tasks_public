# Категория | Имя задачи
Web CTFCoin L3

# Информация
Flask-сервис «проверки цены криптовалюты» формирует SQL-запрос через конкатенацию строк и защищается слабым blacklist-фильтром. Решение — обойти фильтр и через SQLi достать флаг из БД.

# Деплой
```bash
cd deploy
docker compose up -d --build
# сервис будет доступен на http://localhost:5000/
```

# Выдать участникам
Файлы не выдаются (только ссылка на веб-сервис).
