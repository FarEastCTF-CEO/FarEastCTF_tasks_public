Web | CoinBuzz

# Информация
Веб‑сервис на Flask показывает цену выбранной криптовалюты. В обработчике используется конкатенация строк при формировании SQL‑запроса к MySQL (уязвимость SQL Injection в параметре `cryptoname`). Флаг хранится в отдельной базе `customer_secret_data`.

# Деплой
Требуется Docker и docker-compose.

```bash
cd deploy
docker compose up -d --build
```

Сервис будет доступен на `http://<host>:5000/`.

# Выдать участникам
Файлы не выдаются (только ссылка на сервис).

# Описание задачи
Компания CoinBuzz запустила публичную страницу «проверки курса» — выбираешь монету и видишь цену. Похоже, разработчики слишком доверяют пользовательскому вводу.

Найдите способ получить секрет из базы данных.
