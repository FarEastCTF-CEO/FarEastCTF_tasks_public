# Информация
Веб-сервис на Flask для «проверки цены криптовалюты». В обработчике `POST /check_crypto` пользовательский ввод конкатенируется в SQL-запрос к MySQL. Присутствует слабый blacklist по `select/union/from`, который обходится смешанным регистром и комментариями. Флаг хранится в БД (таблица `bank_admin_info`, поле `flag`).
# Деплой
```bash
cd deploy
docker compose up -d --build
```
Сервис поднимется на `http://localhost:5000/`.

Флаг задаётся переменной окружения `FLAG` в `deploy/docker-compose.yml` (можно заменить на нужный).
