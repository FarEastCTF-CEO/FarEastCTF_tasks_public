# web | SQL_L2

# Информация
Серия web-задач про SQL-инъекции (SQL_L2). 
В `deploy/` лежат материалы для развертывания (архив сайта и SQL-дамп/скрипт БД, как в исходном архиве).

# Деплой
Развертывание зависит от содержимого RAR-архива (внутри — файлы сайта/БД).
Типовой сценарий:
1) Распаковать архив из `deploy/`.
2) Поднять веб-сервер/стек, импортировать БД из `deploy/L*.txt`.
3) Проверить, что страница принимает ввод и выполняет SQL-запросы.

(Поскольку RAR не распакован автоматически, конкретные команды смотрите внутри архива после распаковки.)
