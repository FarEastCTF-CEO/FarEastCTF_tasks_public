# stegano | QR

# Информация
В `TASK.png` спрятано сообщение:
- В картинке есть QR-код со ссылкой на алфавит.
- В этой же картинке закодированы данные в виде отдельных цветных пикселей, которые нужно декодировать по алфавиту.

# Деплой
Не требуется (задача файловая).

# Выдать участникам
`public/TASK.png` (и при необходимости `public/ALPHABET.png` для хостинга алфавита)

# Описание для площадки
**QR — «Постер с легендой»**

На постере `TASK.png` есть подсказка, которая ведёт к “легенде” (алфавиту). А сам постер хранит сообщение так, как будто это не текст, а набор сигналов.
Расшифруйте спрятанное послание и заберите флаг.

Формат: CTFKHV{...}


# Флаг
CTFKHV{LFBKZWTM}
