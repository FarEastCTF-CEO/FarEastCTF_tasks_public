# Информация
Дан Python-скрипт с GUI, который просит пароль. При вводе правильного пароля показывает флаг.
Внутри кода есть набор зашифрованных строк и подсказка в Base64 — нужно восстановить пароль.

# Деплой
Не требуется (задача файловая).

# Описание для площадки
**python_reverse — «Не тот пароль»**

Скрипт делает вид, что честно проверяет пароль — но на самом деле он хранит правила проверки внутри себя.
Разберите логику и найдите значение, которое откроет доступ к флагу.

Формат: CTFKHV{...}
