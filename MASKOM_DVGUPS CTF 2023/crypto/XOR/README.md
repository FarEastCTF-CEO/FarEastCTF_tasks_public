# crypto | XOR

# Информация
В `task.txt` две строки:
- первая — шифртекст (hex),
- вторая — строка в Base64.

После Base64-декодирования получается ещё одна зашифрованная строка (простая замена), в которой спрятана подсказка,
как обрабатывать первую строку (XOR + инкремент по модулю 256).

# Деплой
Не требуется (задача файловая).

# Описание для площадки
**XOR — «Тень и маска»**

Иногда секрет прячут не “сложным” шифром, а простой операцией, которая превращает текст в странный набор байтов.
У вас есть данные — выясните, чем их “перекрасили”, и верните флаг в читаемый вид.

Формат: CTFKHV{...}
