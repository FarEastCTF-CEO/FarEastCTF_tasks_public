# Описание для площадки
**ASCII_HEX — «Шестнадцатеричный шёпот»**

На столе лежит клочок бумаги с “рунами” из цифр и букв A–F. Похоже на то, как компьютеры прячут обычные слова от любопытных глаз.
Верните этим символам человеческий вид — и получите флаг.

Формат: CTFKHV{...}
