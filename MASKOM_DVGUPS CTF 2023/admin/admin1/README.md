# admin | admin1

# Информация
Интерактивный сервис. В каждом раунде выдаётся строка в Base64 — нужно отправить её декодированное значение.
Условие по времени: на ответ даётся ограниченное число секунд. Всего нужно пройти все раунды подряд.

# Деплой
Сборка:
- `go build -o admin1 ./src/admin1.go`

Локальный запуск (stdin/stdout):
- `./deploy/admin1`

Запуск как сетевого сервиса (порт 8080):
- `socat TCP-LISTEN:8080,reuseaddr,fork EXEC:"./deploy/admin1",pty,stderr`

В папке `deploy` лежат пример `start.sh` и `Dockerfile`.

# Выдать участникам
Сетевой адрес сервиса (пример): `nc <host> 8080`

# Описание для площадки
**admin1 — «Вахтёр на автопилоте»**

На входе в систему сидит автоматический «вахтёр»: он задаёт короткие проверки одну за другой и не любит, когда отвечают медленно.
Подключитесь к сервису, выдержите все раунды и заберите флаг в финале.

Формат: CTFKHV{...}
