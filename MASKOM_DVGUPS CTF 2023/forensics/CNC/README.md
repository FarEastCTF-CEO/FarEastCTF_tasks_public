# forensics | CNC

# Информация
Дан ELF-файл (`task`). При запуске он выполняет сетевые действия: пытается резолвить домены и подключаться к C2.
Нужно «поймать» то, что он отправляет/получает, и восстановить флаг.

# Деплой
Не требуется (задача файловая). 
Для локальной проверки достаточно Linux-машины (или контейнера) и Wireshark/tcpdump.

# Выдать участникам
`public/task`

# Описание для площадки
**CNC — «Контроллер из цеха»**

В лабораторию принесли подозрительный исполняемый файл: говорят, он общается с “хозяином” и получает команды.
Разберите артефакты общения/поведения и вытащите секрет, спрятанный автором.

Формат: CTFKHV{...}


# Флаг
