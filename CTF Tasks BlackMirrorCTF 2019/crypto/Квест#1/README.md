### Информация
Задание представляет собой многоэтапный квест, сочетающий элементы веб-поиска и классической криптографии. Участнику необходимо последовательно находить скрытые подсказки, переходя от одного источника информации к другому, пока не будет получена ссылка на следующий этап и, в конечном итоге, флаг.

### Деплой
Файлы необходимые для запуска в папке `deploy`
