Для решения этой задачи нам нужно сначала склеить строки в файлах, потому что каждая буква находится на новой строке.
Мы это можем реализовать при помощи вот этого `fish` скрипта:
```fish
for f in *.txt
    tr -d '\r\n' < "$f" > "fixed_$f"
end
```
Далее нам нужно найти учётку `root` и достать из неё пароль, это мы можем сделать при помощи вот этого скрипта:
```fish
for f in *.txt
    tr -d '\r\n' < "$f" | grep -i root
end
```

Готово, получаем флаг!
