Для решения задания использовалась утилита `zsteg` с явным указанием параметров извлечения LSB-данных:
```shell
zsteg -E b1,rgb,lsb,xy LSB.png > dump.bin
```
Расшифровка параметров:
+ b1 — используется первая битовая плоскость;
+ rgb — задействуются все цветовые каналы (R, G, B);
+ lsb — извлекаются младшие значащие биты;
+ xy — порядок чтения пикселей слева направо, сверху вниз.

Затем был выполнен поиск читаемого текста в извлечённом дампе. Так как внутри могли быть текстовые данные, применяем strings:
```shell
strings dump.bin
```
В выводе обнаруживается полезная ссылка. При переходе по ней отображается флаг.
