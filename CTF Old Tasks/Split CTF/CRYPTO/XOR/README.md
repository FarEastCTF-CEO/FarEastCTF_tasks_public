**Информация**  
Участникам предоставляется файл с данными, зашифрованными с помощью операции XOR. Используемая схема намеренно упрощена и позволяет восстановить исходную информацию с помощью анализа и атаки по известному открытому тексту. Задание предназначено для знакомства с принципами работы XOR и базовыми методами его взлома.
