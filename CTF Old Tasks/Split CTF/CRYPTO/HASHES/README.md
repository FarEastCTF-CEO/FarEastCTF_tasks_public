**Категория**  
crypto  

**Название**  
Hex Whispers  

**Информация**  
Задание рассчитано на новичков и предназначено для знакомства с хэш-функцией MD5 и принципами её взлома. Участникам выдаётся список MD5-хэшей, среди которых присутствуют повторяющиеся и общеизвестные значения, легко восстанавливаемые через публичные базы или словарные атаки.   

**Выдать участникам**  
Файл `hashes.txt`  

**Описание для площадки**  
В архиве остались лишь странные шестнадцатеричные отпечатки — одинаковые символы, повторяющиеся снова и снова, будто кто-то уже пытался что-то скрыть, но сделал это слишком просто. Эти следы принадлежат давно устаревшему замку, который помнит каждый свой ключ. Разгадайте, что было скрыто за этими отпечатками, и восстановите истинное послание. SplitCTF{...}.  

**Флаг**  
`SplitCTF{MD5_CR@CKED}`
