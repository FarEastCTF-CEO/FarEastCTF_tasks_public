**Название**  
Old but Gold

**Информация**  
Задание представляет собой классический шифр Цезаря с неизвестным сдвигом, где зашифрованы только латинские буквы, а символы формата флага сохранены. Решение сводится к перебору всех 26 сдвигов, при котором сразу становится виден корректный флаг.

**Выдать участникам**  
Файл `ciphertext.txt`

**Описание для площадки**  
Перед вами задание на классический шифр Цезаря. В файле находится строка, в которой зашифрованы только латинские буквы, а символы формата флага сохранены без изменений. Ваша задача — определить корректный сдвиг и восстановить исходный флаг формата SplitCTF{...}.

**Флаг**  
`SplitCTF{Veni_Vidi_Vici}`
