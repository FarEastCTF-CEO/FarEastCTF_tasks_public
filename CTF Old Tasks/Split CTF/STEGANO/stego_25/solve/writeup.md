# Райтап: «Изнутри мы другие»

## Идея
В JPEG можно дописать любые байты **после** маркера конца изображения `FFD9` (EOI). Большинство просмотрщиков картинок это проигнорируют, но эти данные легко достать утилитами вроде `strings`, `xxd`, `tail`.

## Решение
1. Проверим, нет ли читаемых строк в файле:

```bash
strings Stego_25.jpg | tail -n 10
```

В выводе видна строка с флагом:

```
SplitCTF{y0u_cOnsId3rat3}
```

2. (Опционально) Убедимся, что флаг действительно дописан после EOI:

```bash
# показать последние байты файла
xxd -g 1 Stego_25.jpg | tail

# либо просто прочитать «хвост»
tail -c 64 Stego_25.jpg
```

Вы увидите, что после `... ff d9` (конец JPEG) идут дополнительные байты, среди которых и находится флаг.

## Флаг
`SplitCTF{y0u_cOnsId3rat3}`
