# Writeup: Stego | Stego_75

## Идея
Дан JPEG. Часто в таких задачах «что-то лежит внутри файла» — например, архив, приписанный к концу изображения.

## Решение
1. Проверяем, нет ли внутри архива:

```bash
unzip -l Stego_75.jpg
```

`unzip` выводит предупреждение про «extra bytes at beginning» и при этом показывает содержимое архива — значит, к картинке приписан ZIP.

2. Извлекаем файл из архива прямо из JPG:

```bash
unzip Stego_75.jpg
```

Внутри будет текстовый файл с base64-строкой.

3. Декодируем base64 и получаем флаг:

```bash
# если файл один, можно так:
unzip -p Stego_75.jpg | base64 -d

# либо руками:
cat <извлечённый_файл>.txt | base64 -d
```

`SplitCTF{Th!s_fl@g_truE}`
