# Информация
Веб-страница просит «пароль», который собирается прямо в JavaScript. Нужно восстановить строку и открыть скрытый роут, где лежит флаг.

# Деплой
Собрать и запустить Docker-образ:

```bash
docker build -t lovejs deploy
docker run --rm -p 8080:80 lovejs
```

Сервис будет доступен на `http://localhost:8080/`.

# Выдать участникам
Нет файлов (участникам выдаётся только ссылка на веб-сервис).

# Описание задачи
Мы спрятали флаг за «паролем» в браузере. Говорят, его можно найти, если внимательно посмотреть, что делает JavaScript.
