# Решение

1. Открываем страницу задания и смотрим исходный код (`View Source`).
2. Видим скрипт, который:
   - создаёт массив `base`,
   - переворачивает его (`base.reverse()`),
   - собирает `data[0..7]` из разных индексов,
   - сравнивает ввод с `data.join("_")`.

После `reverse()` индексы указывают на такие символы:

- `base[44] = H`
- `base[11] = o`
- `base[50] = B`
- `base[29] = W`
- `base[61] = 0`
- `base[41] = K`
- `base[49] = C`
- `base[12] = n`

Значит пароль: **`H_o_B_W_0_K_C_n`**.

3. Вводим пароль в prompt — нас редиректит на `/<пароль>` и выдаёт флаг.

## Флаг
`SplitCTF{|0v3_J@v@_$cr!p[_s0_m@c][}`
