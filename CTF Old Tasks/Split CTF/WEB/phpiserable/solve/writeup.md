# Решение

На странице логина видно, что пароль хэшируется MD5 и сравнивается так:

```php
hash('md5', $pwd, false) == "0e111111111111111111111111111111"
```

Ключевой момент — используется **нестрогое сравнение `==`**. В PHP строки вида `0e12345...` при сравнении могут приводиться к числу в научной записи (как `0 * 10^...`), то есть к нулю.

Значит достаточно подобрать любой пароль, чей MD5 имеет вид `0e` + только цифры.

Один из известных примеров:

- пароль: `240610708`
- `md5("240610708") = 0e462097431906509019562988736854`

Обе строки (`0e...`) при `==` будут равны нулю, и проверка пройдёт.

## Эксплуатация
1. Вводим `login=admin`
2. Вводим `password=240610708`
3. Получаем содержимое `/flag.txt`.
