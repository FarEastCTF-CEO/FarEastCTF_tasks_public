# Информация
Уязвимость type juggling в PHP: сравнение MD5-хэша через `==` позволяет обойти проверку «магическим» хэшем формата `0e...`.

# Деплой
Собрать и запустить Docker-образ:

```bash
docker build -t phpiserable deploy
docker run --rm -p 8080:80 phpiserable
```

# Выдать участникам
Нет файлов (участникам выдаётся только ссылка на веб-сервис).

# Описание задачи
Хотите немного магии? PHP-скрипт ждёт от вас ввода МАГИЧЕСКОГО пароля. Имейте в виду, что пароли хэшируются.
