# Forensics | Boris (USB HID)

# Информация
Дан `pcap`, снятый **USBPcap** в Windows. Внутри находятся **HID-репорты клавиатуры** (USB interrupt transfer),
по которым можно восстановить введённый текст. В этом тексте присутствует флаг в формате `SplitCTF{...}`.

Подробное решение — в `solve/writeup.md`.

# Описание для площадки
Вам дан дамп USB-трафика (USBPcap). Восстановите нажатия клавиатуры из HID-репортов и найдите флаг формата `SplitCTF{...}`.
