# Writeup: «Барри»

В `public/Swift.txt` лежит набор пар чисел `a,b`.
В `public/book/` лежат 24 главы книги **The Swift Programming Language**, пронумерованные `01_...` — `24_...`.

Идея задачи — **книжный шифр** + подсказка из названия платформы (**SplitCTF**): нужно получить две строки, разделив результат.

## 1) Достаём слова по координатам

Для каждой строки `a,b`:

1. Открываем главу №`a` (файл `public/book/{a:02d}_*.md`).
2. Разбиваем текст главы на слова. В задаче под словом понимается последовательность латинских букв (регэксп):

```regex
[A-Za-z]+
```

3. Берём `b`-е слово (нумерация с 1).

После этого получаем 36 слов.

## 2) Берём первые буквы

Из каждого найденного слова берём первую букву и склеиваем — получаем строку из 36 букв.

## 3) Split: нечётные и чётные позиции

Дальше делаем «split» по позициям:

- **нечётные** символы (1‑й, 3‑й, 5‑й, …) → одна строка
- **чётные** символы (2‑й, 4‑й, 6‑й, …) → вторая строка

Одна строка будет осмысленной и даст тело флага:

- odd → `OCChoiseIndividual`
- even → `EvenLettersDecoyOk` (приманка)

Значит флаг:

`SplitCTF{OCChoiseIndividual}`
