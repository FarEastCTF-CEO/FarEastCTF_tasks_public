# Райтап: Капитан Очевидность

## Идея
Участникам выдаётся файл `Task.js` — это JavaScript, обфусцированный в стиле **aaencode** (код начинается с символов `ﾟωﾟﾉ=`).
При выполнении скрипт много раз вызывает `alert(...)`: в большинстве случаев показываются случайные строки, но периодически выводятся элементы массива `flag_here`, среди которых спрятаны две части флага.

## Проверка работоспособности
Если выполнить `Task.js`, можно увидеть, что среди выводимых строк встречаются:

- `SplitCTF{iTs`
- `_aWEs0mE!}`

Их конкатенация даёт флаг.

## Решение (вариант через запуск в Node.js)
1. Создаём небольшой раннер, чтобы заменить `alert` на вывод в консоль:

```js
global.alert = (x) => console.log(String(x));
require('./Task.js');
```

2. Запускаем и фильтруем вывод:

```bash
node run.js | grep -E 'SplitCTF\{|aWEs0mE'
```

3. Склеиваем две части:
`SplitCTF{iTs` + `_aWEs0mE!}` → **SplitCTF{iTs_aWEs0mE!}**

## Решение (вариант через анализ кода)
1. Деобфусцируем `Task.js` (например, любым aaencode-деобфускатором или просто выполняем скрипт и смотрим, что он делает).
2. В раскрытом коде видно массив `flag_here`, где есть строки `SplitCTF{iTs` и `_aWEs0mE!}`.
3. Соединяем их и получаем флаг: **SplitCTF{iTs_aWEs0mE!}**
