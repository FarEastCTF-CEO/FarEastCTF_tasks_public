# negative — writeup

## Идея
`int32_t` переполняется по модулю `2^32`.

Даны:
- `a` (положительное)
- `b` (отрицательное)

Нужно ввести **положительное** `x` (вводится как `uint32_t`), чтобы выполнялось:

```
(int32_t)(a + x) == b
```

## Решение
В арифметике по модулю `2^32`:

```
x ≡ (b - a) (mod 2^32)
```

То есть:

```python
x = (b - a) & 0xffffffff
```

## Эксплуатация
1) Подставляем числа из вывода программы.
2) Считаем `x`.
3) Отправляем его.

Пример (локально):

```bash
./negative
# a = ...
# b = ...
python3 - <<'PY'
a = int(input('a='))
b = int(input('b='))
print((b - a) & 0xffffffff)
PY
```

Для `nc` — то же самое: посчитать `x` и отправить числом.
