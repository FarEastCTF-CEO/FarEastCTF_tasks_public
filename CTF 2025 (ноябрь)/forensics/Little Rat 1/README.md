# forensics | Little Rat 1

# Информация
Участникам дан журнал security событий Windows, `little_rat.evtx`.

Ключ реестра `Flag` в разделе `Environment` содержал флаг в Base64. Пейлод заменил его на фальшивый.

Ключевое событие для решения — ID 4657 (The registry value was modified),
которое в своих данных содержит как старое (исходный флаг в base64), так и новое (флаг-заглушка, с открытым тегом CTF и самим значением флага в base64) значения.
Во флаге-заглушке специально тег оставлен открытым, чтобы привлечь внимание.
