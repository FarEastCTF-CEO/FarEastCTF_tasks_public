# Admin | QR Tool
# Деплой
В папке deploy выполняем команду, предварительно указав нужные порты в docker-compose.yml:
```shell
docker compose up -d --build
```

# Выдать участникам
Команду для подключения:
```shell
nc -N localhost 10003
```

Подсказки:
1. Кажется, у сервиса есть минусы, которые могут вам помочь
2. Флаг где-то в окружении
