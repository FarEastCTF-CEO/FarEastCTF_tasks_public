# Информация
1) Генерируем с помощью burp или jwt_tool пару RSA ключей
2) Публикуем у себя на сервере jwks.json, например с помощью: python3 -m http.server <port>
3) Меняем user_id, подставляем в заголовок параметр jku и подписываем токен сгенерированным приватным ключом
4) Далее для визуального отображения ответа берем токен из запроса и подставляем его на ручку /content и получаем флаг
