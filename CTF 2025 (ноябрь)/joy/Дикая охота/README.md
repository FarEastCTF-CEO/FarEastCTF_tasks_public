# Описание для площадки
Здание на подебажить, пореверсить и покопаться в солюшенах. Первая часть флага найдётся при поимке покемона Gastly. Вторая часть флага (4 цифры) будет видна на локации, где Gastly можно впервые встретить.
Формат флага: FECTF{ЗАГЛАВНЫЕ_ЛАТИНСКИЕ_БУКВЫ_ЧЕТЫРЕ_ЦИФРЫ}
