# Информация
Задание на реверс dll / анализа файла в песочнице / анализа файла на виртуалке.
Dll написана на c++, обфускации/шифрования в коде нет. Необходимо определить ресурс, куда идут обращения с хоста жертвы после запуска dll.
После поиска этого домена в файле с выгрузкой из TI системы, становится понятно, какой группировке принадлежит данная "активность".

# Описание для площадки
Был перехвачен подозрительный файл malware_dll.dll, обнаруженный на заражённой рабочей станции. В окрестности инцидента на хосте сработала сигнатура антивируса Trojan-Spy.Win32.KeyLogger.gen. Ваш соратник по вирусной аналитике предположил, что с помощью данной библиотеки злоумышленники загружали кейлоггер на хост жертвы. Попробуйте разгадать загадку библиотеки! И определите, к какой группировке можно отнести полученные улики, с помощью данных, выгруженных из TI-системы.

Формат флага: FECTF{имя группировки}
