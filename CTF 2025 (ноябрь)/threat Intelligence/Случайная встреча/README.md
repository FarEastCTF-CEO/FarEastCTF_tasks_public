# Описание для площадки
Вы — аналитик ИБ в крупном холдинге. В рамках расследования инцидента с подозрение обнаружения индикатора компрометации Threat Intelligence вам предоставили выгрузку из SIEM-системы.
Единственное, что известно – один из сотрудников отдела разработки в рабочее время посещал сайт hxxps://mottogroup[.]in/ компании, занимающейся продажей керамической плитки. Вам необходимо определить, какая Zero-Day уязвимость в системе сотрудника могла бы быть использована злоумышленниками, если бы этот визит был частью целенаправленной атаки.
