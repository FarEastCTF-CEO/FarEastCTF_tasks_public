# crypto | Task 1

# Информация
Флаг зашифрован так же, как и в Android FDE, необходимо расшифровать его, имея мастер ключ  от устройства. 

# Выдать участникам
На площадку выкладывается архив ../public/Task.zip.

# Описание для площадки
Вам необходимо расшифровать дамп памяти Android устройства и извлечь из него флаг. Дамп зашифрован при помощи Android FDE. Для этого, у вас имеются следующие данные:
- master_key.bin - мастер ключ от устройства.
- encrypted_data.dump - непосредственно дамп памяти. Дамп начинается с сектора с номером 324.
- crypt_mnt_ftr.bin - параметры шифрования. Описание crypt_mnt_ftr можно посмотреть здесь https://android.googlesource.com/platform/system/vold/+/refs/heads/kitkat-dev/cryptfs.h
P.s. размер сектора - 512 байт.
