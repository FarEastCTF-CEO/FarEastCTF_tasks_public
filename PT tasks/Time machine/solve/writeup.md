# Time machine - writeup
Рассмотрим код и поймем, что этот код реализует виртуальную машину (VM), которая принимает строку ввода и выполняет серию операций над регистрами, сравнивая значения с определенными константами и значениями, загруженными из входной строки. В результате проверяется, соответствует ли входная строка определенному шаблону.

У нас есть следующие функции:

add: складывает значения двух регистров и сохраняет результат в третьем регистре.
sub: вычитает значение одного регистра из другого и сохраняет результат в третьем регистре.
mul: умножает значения двух регистров и сохраняет результат в третьем регистре.
load: загружает ASCII-код символа из входной строки по индексу в регистр.
load_const: загружает константу в регистр.
comp_critical: сравнивает значения двух регистров и устанавливает счетчик пропусков, если они не равны.
set_ok: устанавливает флаг ok в значение True.
Для решения таска нужно поэтапно разобрать строчку на 87 строке.

Для примера разберем первые 3 символа:
1) load_const('r6', 86) - Задать r6 = 86 (константа 86).

2) load('r5', 0) - Загрузить ASCII-код первого символа входной строки в регистр r5.

3) comp_critical('r5', 'r6') - Сравнить значение регистра r5 с r6. Если они не равны, установить self.skip = 0xdead.

Значит ASCII код первого символа должен быть равен 86, первый символ V

load_const('r1', 23) - Задать r1 = 23 (константа 23).
add('r6', 'r6', 'r1') - Сложить значения r6 и r1 и сохранить результат в r6 (r6 = r6 + r1).
load('r5', 1) - Загрузить ASCII-код второго символа входной строки в регистр r5.
comp_critical('r6', 'r5') - Сравнить значение регистра r6 с r5. Если они не равны, установить self.skip = 0xdead
Значит ASCII код второго символа должен быть равен 23+86=109, второй символ m
