# Permutations - writeup
В данной задаче необходимо было использовать двумерный массив для построения цепочки длиной 64. Процесс решения задачи начинался с анализа бинарного файла в IDA, где был обнаружен двумерный массив, играющий ключевую роль. После извлечения массива с помощью инструментов IDA было установлено, что каждый его элемент можно использовать для построения цепочки, соблюдая определенные условия.

Для построения цепочки был разработан рекурсивный алгоритм. Изначально выбирался первый элемент из массива. Затем каждый новый элемент, добавляемый к цепочке, должен был удовлетворять условиям, связанным с масками мьютексов. После добавления нового элемента к цепочке обновлялась маска мьютексов, чтобы учитывать новые ограничения. Из оставшихся элементов выбирались те, которые соответствовали обновленной маске мьютексов, и добавлялись к цепочке. Этот процесс повторялся рекурсивно до тех пор, пока не удавалось построить цепочку длиной 64.
