# Magic word - writeup
Проанализировав исходный код, поймем, что:
Длина входной строки должна составлять 11 символов.

Хэш, рассчитанный по первым 4 байтам входных данных, должен равняться 0x1114f2dff.

Хэш, рассчитанный на основе следующих 4 байт входных данных, должен равняться 0x10b421402.

Хэш, рассчитанный на основе следующих 3 байтов входных данных, должен равняться 0xcb431201.

Чтобы найти корректный ввод, вычисляем исходные значения ввода, которые после применения операций дадут нужные хэши. Для этого вычитаем константу 0xcafedead из целевых хэшей и преобразуем результат в строку.

Пример кода лежит в файле `solver.py`

флаг: `FECTF{ROPFU5C@T3D}`
